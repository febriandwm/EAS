Repository ini dibuat untuk memenuhi tugas Evaluasi Akhir Semester (EAS) mata kuliah **Pemelajaran Mesin (Machine Learning)**. 
Proyek ini berfokus pada klasifikasi tingkat kepadatan lalu lintas menggunakan algoritma **Random Forest Classifier** dan dideploy menjadi aplikasi web dengan **Streamlit**.

Sumber dataset: [Kaggle - Traffic Prediction Dataset](https://www.kaggle.com/datasets/hasibullahaman/traffic-prediction-dataset)
